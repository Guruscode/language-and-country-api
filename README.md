# Language and Country API

This API provides a list of languages and countries using data from the RestCountries API.

## Endpoints

### Get All Languages

- **URL:** `/api/languages`
- **Method:** `GET`
- **Response:** Array of all languages.

### Get All Countries

- **URL:** `/api/countries`
- **Method:** `GET`
- **Response:** Array of all country names.
